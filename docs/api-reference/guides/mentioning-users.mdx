---
title: 'Mentioning users'
---

To mention a user in a post, comment, or message, use `<@member_id>` in the content. Example:

```json
{
  "title": "Daily standup",
  "content_markdown": "What did you work on today? cc <@3hvr0vv6fb75>, <@zwvznmxcpvel>",
  "channel_id": "7eokvf4jh5mr"
}
```

To get a teammate’s member ID:

1. Click your organization name in the top left to open the dropdown.
2. Click **People**.
3. Click the **•••** menu next to a user.
4. Click **Copy member ID**.
