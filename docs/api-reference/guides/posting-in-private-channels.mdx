---
title: 'Posting in private channels'
---

Integrations can post in any public channel, but require additional permissions to post in a private channel. Here’s how to grant access:

1. In a private channel, click the **•••** button in the title bar, then click **Integrations**.
2. Click **Add to channel** next to the integration(s) you want to allow in that channel.

To get a channel’s ID, click the **•••** button in the title bar, then click **Copy ID**.
